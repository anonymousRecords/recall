# Recall
**Where memory becomes skill**

**Recall**은 취준생 개발자를 위한 코딩 실력 유지 도구입니다.

알고리즘 문제를 풀고 잊어버리는 반복을 끊고, AI 면접관과의 실전 연습을 통해 기억을 진짜 실력으로 전환합니다.

<br />

<h3>1. 에빙하우스 망각곡선 기반 복습 스케줄링</h3>

풀고 나서 그냥 넘어가면 며칠 안에 잊어버려요.   

Recall은 에빙하우스 망각곡선 이론에 따라 **1 → 3 → 7 → 14 → 30일** 간격으로 복습 시점을 자동 계산해 오늘 풀어야 할 문제를 대시보드에 모아줍니다.

- 문제를 등록하면 다음 복습일이 자동 지정
- 복습 완료 시 다음 단계로 넘어가고, 진행 상태를 시각화
- 복습 주기는 설정에서 커스터마이징 가능

<h3>2. AI 모의 개발 면접</h3>

프로그래머스에서 문제를 열면 Recall이 문제 정보를 자동으로 감지해요.

AI 면접관과 **실시간으로 문제 풀이 과정을 설명**하며 실전 면접을 시뮬레이션 해줘요.

- 면접관 스타일 선택 가능: 친절 / 보통 / 압박
- 제한 시간 설정: 30분 / 45분 / 60분 / 무제한
- 면접 종료 후 4가지 항목 점수와 AI 피드백 제공
- 문제 이해 · 소통 능력 · 코드 품질 · 시간 관리
- AI 제공자 선택 가능: OpenAI GPT-4o-mini / Claude Haiku

<h3>3. 면접 기록 및 AI 비용 추적</h3>
면접 이력을 누적하면 내 성장 흐름과 AI 사용 비용을 한눈에 볼 수 있어요.

- 점수 추이 그래프로 실력 변화 확인
- 면접관 스타일별 · AI 프로바이더별 평균 점수 비교
- 세션당 토큰 소비량 및 누적 비용 추적 (`$0.0000` 단위)
- 기간 필터: 최근 10회 / 최근 30일 / 전체
